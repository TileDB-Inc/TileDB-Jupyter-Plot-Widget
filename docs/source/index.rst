
tiledb_plot_widget
=====================================

Version: |release|

Custom Jupyterlab widget for TileDB


Quickstart
----------

To get started with tiledb_plot_widget, install with pip::

    pip install tiledb_plot_widget

or with conda::

    conda install tiledb_plot_widget
